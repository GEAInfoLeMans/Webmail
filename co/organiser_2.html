Filtres de courriers

Un filtre de courrier permet de ranger automatiquement les courriers dans des dossiers. Les courriers contenant une expression sont classés dans un dossier. L'expression peut être une adresse mail, un mot clef dans la partie objet ou le corps du message.

Par exemple dans la vidéo suivante un filtre est crée permettant de classer les courriers venant d'un expéditeur dont le nom contient G11 dans le sous dossier G11.
